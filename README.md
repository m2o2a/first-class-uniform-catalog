# First Class For Uniform — Digital Catalog

كتالوج إلكتروني Premium لشركة First Class For Uniform. مبني بـ Next.js 14 + TypeScript + Tailwind CSS.

## تشغيل المشروع محليًا

```bash
npm install
npm run dev
```

افتح http://localhost:3000

## البناء للنشر (Production)

```bash
npm run build
npm start
```

الأفضل للنشر: Vercel (اسحب المجلد أو اربط الـ Git repo — بيشتغل من غير أي إعدادات إضافية).

## هيكل المشروع

```
app/
  page.tsx                → الصفحة الرئيسية (Hero + أقسام مختارة + سابقة أعمال + CTA)
  sections/page.tsx       → كل الـ 22 قسم
  sections/[slug]/page.tsx→ صفحة كل قسم (Banner + Gallery + CTA)
  projects/page.tsx       → سابقة الأعمال حسب القطاع
  clients/page.tsx        → شعارات العملاء
  contact/page.tsx        → واتساب / اتصال / مشاركة الكتالوج
components/               → أجزاء الواجهة القابلة لإعادة الاستخدام
lib/
  categories.ts           → قائمة الـ 22 قسم (عدّل هنا لو غيّرت الأقسام)
  sectors.ts              → قطاعات سابقة الأعمال
  dictionary.ts           → كل النصوص عربي/إنجليزي في مكان واحد
  contact.ts              → أرقام التليفون ورابط الواتساب
```

## إضافة صور المنتجات الحقيقية (تلقائي — بدون تعديل كود)

كل قسم بيحاول يعرض 8 صور من المسار ده:

```
public/products/{slug}/1.jpg
public/products/{slug}/2.jpg
...
public/products/{slug}/8.jpg
```

مثال لقسم الأمن: `public/products/security/1.jpg`. الأسماء (slugs) كلها موجودة في `lib/categories.ts`.

**الووترمارك بيتحط تلقائي.** أي صورة تحطها في المسار ده، اللوجو بتاعك (`public/logo/logo.png`) هيظهر فوقها في الزاوية أوتوماتيك — مش محتاج تعدل الصورة بنفسك أو تستخدم فوتوشوب.

لو المسار فاضي (لسه مبعتلك مافيش صورة)، الموقع بيرجع لكارت "قريبًا" أنيق بدل ما يبوّظ التصميم.

### من فين تجيب صور مؤقتة (قانونية 100%)

⚠️ لا تنزل صور من نتائج جوجل العادية أو من مواقع شركات تنافسك — دي محمية بحقوق نشر ومش ملكك.

استخدم مكتبات صور مجانية للاستخدام التجاري:
- https://unsplash.com
- https://pexels.com
- https://pixabay.com

ادخل، اكتب كلمة زي "security guard uniform" أو "chef uniform"، اختار صورة عامة (من غير لوجو شركة تانية ظاهر فيها)، نزّلها، وضيفها في مكانها بالمسار فوق.

## رفع الموقع على الإنترنت (Deploy)

أسهل وأسرع طريقة هي **Vercel** (الشركة اللي عملت Next.js نفسه — مجاني للمشاريع الصغيرة زي ده):

### الطريقة 1 — من غير Git (أسرع، دقيقتين)
1. روح على https://vercel.com واعمل حساب مجاني (تسجيل بإيميلك أو بحساب جوجل)
2. من الداشبورد اضغط **Add New → Project**
3. اختار **Deploy without Git** / أو اسحب مجلد المشروع كامل (بعد `npm install` مرة واحدة عندك) وارفعه
4. Vercel هيتعرف على المشروع إنه Next.js تلقائي ويبنيه وينشره
5. هيديك رابط زي `first-class-uniform.vercel.app` تقدر تبعته لعملائك فورًا

### الطريقة 2 — عن طريق GitHub (أفضل لو هتعدّل باستمرار)
1. اعمل مستودع (repo) جديد على https://github.com وارفع عليه المشروع
2. من Vercel: **Add New → Project → Import Git Repository** واختار المستودع
3. اضغط **Deploy** — وبعد كل تعديل تبعته لـ GitHub، الموقع بيتحدث تلقائي من غير ما تعمل حاجة

### ربط دومين خاص بيك (اختياري)
بعد ما ترفع، من إعدادات المشروع في Vercel اختار **Domains** وضيف الدومين بتاعك (مثلاً `firstclassuniform.com`) واتبع تعليمات الـ DNS اللي هيوريها لك.

لو حابب أساعدك تجهز حساب Vercel وتربطه، قولي وهمشي معاك خطوة بخطوة.

## اللغة

الموقع يبدأ بالعربي افتراضيًا (RTL) مع زرار تبديل للإنجليزي في الهيدر. كل النصوص في `lib/dictionary.ts`.

## ملاحظة عن شعارات العملاء

صورتين سابقة الأعمال اللي بعتهم اتحطوا زي ما هما في `public/clients/`. لو حابب كل شعار يبقى كارت منفصل بدل الصورة الكاملة، محتاج تبعتلي كل لوجو كملف مستقل وهظبط الشبكة تلقائي.
